4.4.4. Линейные модели экономики

Линейная модель экономики может быть записана как СЛАУ x - Ax = y, где элементы матрицы A и столбца y — неотрицательные числа.

1\. Используя определение продуктивности, проверьте, являются ли матрицы продуктивными:

a) \[1 2; 3 4\]

b) (1/2) \* \[1 2; 3 4\]

c) (1/10) \* \[1 2; 3 4\]

2\. Используя критерий продуктивности ((E - A)^(-1) имеет только неотрицательные элементы), проверьте, являются ли матрицы продуктивными:

a) \[1 2; 3 1\]

b) (1/2) \* \[1 2; 3 1\]

c) (1/10) \* \[1 2; 3 1\]

3\. Используя спектральный критерий продуктивности, проверьте, являются ли матрицы продуктивными:

a) \[1 2; 3 1\]

b) (1/2) \* \[1 2; 3 1\]

c) (1/10) \* \[1 2; 3 1\]

d) \[0.1 0.2 0.3; 0 0.1 0.2; 0 0.1 0.3\]
